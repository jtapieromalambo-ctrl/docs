Signia is a Django 5.2 web application that bridges spoken/written Spanish and Colombian Sign Language (LSC). It bundles a text/audio-to-sign translator, a camera-based sign-to-text recognizer, user authentication with OAuth, and an admin panel for managing the sign-video library and retraining the recognition model.

Use these docs to get Signia running on your machine for development, or to understand the moving parts before deploying it.

## What's inside

- **`traduccion/`** — text and audio to LSC video translation, backed by `faster-whisper` for speech-to-text and Groq for LSC grammar conversion.
- **`reconocimientos/`** — camera-based sign recognition using MediaPipe HandLandmarker and a RandomForest classifier.
- **`usuarios/`** — custom user model, OTP email verification, Google/Facebook OAuth via `django-allauth`, and the Brevo email backend.
- **`historial/`** — per-user history of translations and recognitions.
